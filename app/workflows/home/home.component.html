MEAN Material Skeleton Application

This is a bare-bones solution to help you get started with building a MEAN stack application, without the Angular CLI. It includes:

  • MEAN Stack
  • Angular 6
  • Angular Flex Layout
  • Angular Material
  • Webpack

Technical

Technically speaking, this application is pre-coded with a shared module, lazy loading, and supervisor (for watching files) already set up for you. In addition, a number of pre-defined convenience components are included, such as a loading indicator.

Server-side, you will find the folder structure already defined for you. Some basic routing is provided for you, as well as a module loader. There is also an environment-dependent configuration a la Zend Framework, and several sample environment configs are provided.

Most importantly, the skeleton comes pre-configured with both JIT and AOT compilation.
